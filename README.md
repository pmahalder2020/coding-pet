# MAGI SYSTEM — Pilot Sync Monitor

> *God's in his heaven. All's right with the world.*

A Neon Genesis Evangelion-themed coding pet that evolves as you code. Log sessions, complete missions, and watch your EVA unit grow from a dormant entry plug into a godlike being.

---

## Features

- **Live countdown timer** — pick 10, 15, 30, 60, or 90 minutes. XP is awarded on completion, scaling with duration.
- **Mission objectives** — check off what you built or solved each session for bonus XP.
- **6 evolution stages** — your unit grows from a sealed pod (with escort ships) through a bipedal mech, berserk mode, S² engine activation, all the way to ADAM.
- **Unit colour picker** — choose your Eva unit colour: Unit-01 purple, Unit-02 red, Unit-00 blue, and more.
- **Deployment calendar** — visual month view with coded days marked, plus tally marks and a streak progress bar toward milestones.
- **NERV terminal aesthetic** — orange-on-black MAGI panels, CRT scanlines, circuit grid background, cyan readouts.
- **Two-column layout** — EVA status pinned on the left, controls on the right.
- **Fully offline** — single HTML file, no dependencies, no server.

## Usage

Download `index.html` and open it in any browser. Progress is saved in local storage.

To host it live, push to GitHub and enable **Settings → Pages → main branch**. Your pet will be accessible at `yourname.github.io/repo-name` from any device.

---

## Stack

Vanilla HTML, CSS, JavaScript. Single file. No frameworks, no build step, no internet required.
